2 opciones (ambas requieren instalar python):
1- Instalar las librerías necesarias para que funcione el script (utilizando pip install) y correr el archivo __main__.py.
2- Ejecutar el siguiente comando en la terminal:
    pip install pyinstaller
    (reiniciar la terminal)
    pyinstaller --onefile --windowed __main__.py

Si se elige la opción 2, se creará una carpeta dist con el ejecutable del programa.

Si la opcion 2 da algun problema de "Path", se puede usar la opcion 1 o bien reiniciar la PC e intentarlo nuevamente.
